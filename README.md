# JARVIS

JARVIS is a voice-controlled AI assistant built in Python.

## Features

- Voice conversations
- Text mode
- Multi-model support
  - Gemini
  - GPT
  - DeepSeek
- Real-time speech recognition
- ElevenLabs text-to-speech
- Automation capabilities
- Model fallback system

## Tech Stack

- Python
- OpenRouter
- ElevenLabs
- RealtimeSTT

- Tool calling
